## Overview

Building an insurance digital platform for customers to get motor car or motorcycle insurance covers and claims. 

This digital platform also provides an option for paying for the insurance covers in installments.

## Technologies Used

- **VueJS**: For building the web application, using the component-based architecture .
- **PrimeVue**: The feature-rich UI component library for styling the components.
